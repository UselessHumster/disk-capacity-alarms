# Disk Capacity Alarms Script

### Бот берет данные из строки df-h, парсит их и присылает в телеграм


При первом запуске происходит настройка бота
Скрипт запросит:
- токен
- ID чата куда слать данные
- какие диски или дирректории проверять

Установка:

```
pip install uv
uv sync
uv run main.py
```
